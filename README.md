## вљ™пёЏ How It Works

1. The user logs daily activities such as travel, electricity use, or food consumption.
2. The frontend sends the activity data to the backend using **HTTP requests**.
3. Spring Boot processes the request and calculates estimated **carbon emissions**.
4. The data is stored in the **MongoDB database**.
5. Users can view their overall carbon footprint and track their environmental impact.
